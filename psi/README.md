psi-lang
==============

- A toy language
- Everything ends in semicolon
- Has strong types
- Has function declaration
- Has pipes
- Expressions evaluated from right to left
- No array, map, or decimals



#### Variable declaration
- `x = 2;`
- `x = 2 + 3;`

#### Function declaration
- `y = (x) => { x + 2; };`
- `print(y(x));`
- `1|>print`
